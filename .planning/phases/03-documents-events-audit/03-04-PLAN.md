<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| AccessFilter to REST output | Incorrect filtering can expose metadata or raw files. |
| Java to MinIO presigned URL | URL issuance grants temporary direct object access. |
| Delete request to indexing cleanup | Missing delete event leaves stale retrieval data later. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04-01 | Information Disclosure | document detail/raw | mitigate | Return 404 for invisible documents and share repository visibility logic. |
| T-03-04-02 | Information Disclosure | presigned raw URL | mitigate | Issue URL only after visibility check, TTL 5 minutes, audit issuance. |
| T-03-04-03 | Tampering | delete lifecycle | mitigate | Soft delete and outbox event in one transaction; no physical object deletion in Phase 3. |
| T-03-04-04 | Repudiation | delete/raw actions | mitigate | Audit sensitive raw URL issuance and deletes with correlation ID. |
</threat_model>
