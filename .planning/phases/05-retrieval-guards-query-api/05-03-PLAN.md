## Trust Boundaries

| Boundary | Description |
|---|---|
| Java access filter to Qdrant query | Filter must be applied inside Qdrant before payloads return. |
| Qdrant payload to generation | Retrieved text may include flagged sanitized chunks. |
| Embedding model to query path | Local model failure must fail loud and not fake recall. |
