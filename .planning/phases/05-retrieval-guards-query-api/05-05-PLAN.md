<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| Retrieved child chunks to LLM context | Parent expansion can add text beyond the child match. |
| Local reranker to request latency | Cross-encoder scoring is heavyweight and shared. |
| Citation IDs to source viewing | Parent/child ID confusion can show the wrong source. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-05-05-01 | Information Disclosure | parent expansion | mitigate | Resolve only parents referenced by already access-filtered child candidates. |
| T-05-05-02 | Denial of Service | reranker concurrency | mitigate | Semaphore size 1, final top-N caps, and disabled fallback. |
| T-05-05-03 | Tampering | citation mapping | mitigate | Preserve child UUIDs as the only citation IDs and test `[N]` mapping. |
</threat_model>
